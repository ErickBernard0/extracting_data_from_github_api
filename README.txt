# GitHub Repositories Explorer

## Descrição
Este projeto permite explorar os repositórios de um usuário no GitHub, extraindo informações como nome, descrição, linguagem, data de criação e data de atualização. As informações podem ser salvas em um arquivo CSV e visualizadas através de uma interface simples feita com Streamlit.

## Tecnologias Utilizadas
- Python
- Pandas
- Requests
- Streamlit
- dotenv
